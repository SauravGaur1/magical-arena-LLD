# Swiggy Magical Arena

## Description || Rules
It simulates a magical arena where players battle based on their health, strength, and attack attributes. Players attack and defend using dice rolls, and the game ends when a player's health reaches zero.

## Installation
To install the dependencies, run the following command in your terminal:
```
npm install
```
```
Logger Service
A flexible logger service that uses strategies pattern :

To Console: Logs messages to the console.
To File: Logs messages to a file located in src/services/loger/data.
```

## Usage
To start the application in development mode:
```
npm run startDev
```

To start the application in production mode (runs tests and then starts):
```
npm run startProd
```

## Tests
Tests for the application can be executed using:
```
npm test
```

## Dev-Dependencies for Testing
- `chai`: Assertion library for testing.
- `mocha`: Test framework.
- `sinon`: Test spies, stubs, and mocks.
